# YTA-project
## ⚡ Electricity Cost & Energy Consumption Prediction Model

### 📌 Overview
This project presents a machine learning model that predicts both **electricity cost** and **energy consumption** based on real-world features such as building characteristics, location, and usage patterns.  
It extends the previous model by providing a more comprehensive analysis of energy behavior.

---

### 🎯 Objective
To develop a **multi-output regression model** capable of simultaneously predicting:
- 💡 Electricity Cost  
- ⚡ Energy Consumption  

This enables better understanding, monitoring, and optimization of energy usage.

---

### 📊 Dataset Description
The dataset includes realistic electricity data with features such as:
- 🏢 Building type and size  
- 📍 City zone / location  
- ⏰ Time-related factors (e.g., peak hours)  
- 🌡️ Environmental and usage conditions  

**Targets:**
- Electricity Cost  
- Energy Consumption  

---

### ⚙️ Methodology
The model follows a standard ML pipeline:

1. **Data Preprocessing**
   - Handling missing values  
   - Encoding categorical variables  
   - Feature scaling  

2. **Model Training**
   - Multi-output regression approach  
   - Train-test split for evaluation  

3. **Evaluation**
   - Metrics such as MAE, MSE, and R² used to assess performance  

---

### 🚀 Key Features
- Predicts **two outputs simultaneously** (cost & consumption)  
- Handles mixed data types (categorical + numerical)  
- Designed for scalability and real-world applications  

---

### 📈 Use Cases
- 🏠 Smart home energy monitoring  
- 🏢 Building energy management systems  
- ⚡ Electricity demand forecasting  
- 📊 Integration with IoT-based energy tracking  

---

### 📬 Conclusion
This model demonstrates how machine learning can provide **deeper insights into electricity usage**, helping users predict both consumption and cost for smarter energy decisions.
